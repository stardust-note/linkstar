# LinkStar

**LinkStar**는 별먼지일기에서 개발한 프로젝트들과 작업물들의 링크를 한 곳에 정리해두기 위한  
간단한 링크 모음 페이지입니다. Linktree와 비슷한 형태로, 앞으로 개발할 사이트들을  
편하게 관리하고 공유하기 위해 제작되었습니다.

---

## 📌 목적

- 개발한 프로젝트/사이트 링크를 한눈에 확인  
- 포트폴리오 및 개인 작업물을 정리하는 허브 역할  
- 여러 링크를 하나의 페이지에서 효율적으로 관리

---

## ✨ 주요 기능

- 프로젝트 및 사이트 링크 목록 제공  
- 간단하고 직관적인 UI  
- 모바일/PC 반응형 지원  
- 가벼운 HTML/CSS 기반 페이지

---

## 🛠 Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **Responsive Web**
